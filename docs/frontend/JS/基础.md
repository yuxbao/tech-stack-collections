### Proxy

[Proxy 官方文档](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Proxy)

`Proxy`本质上是代理模式的实现，创建一个代理的类代替原始类操作。

```js
const target = {
  message1: "hello",
  message2: "everyone",
};

const handler2 = {
  get(target, prop, receiver) {
    return "world";
  }

  return Reflect.get(...arguments);
};

const proxy2 = new Proxy(target, handler2);

console.log(proxy2.message1); // world
console.log(proxy2.message2); // world
```

handler 定义哪些方法会被代理类接管以及如何接管。example 接管了对象属性的`get`操作。

Reflect 方法仍然通过对象的内部方法与对象进行交互,但它们是作为函数调用的，而不是作为对象的方法调用的。

Vue 的响应式实现用了 Proxy，详情可以看[vue 基础语法](../vbtddbwp/)，比如 reactive()返回的还是一个 Proxy
对象，只有代理对象才是响应式的。
